```Java title:loops.java
public class loops {
	public static void main(String[] args) {
		
		System.out.println("While Loop:");
		int i = 0;
		while (i < 5) {
			System.out.println(i);
			i++;
		}
		
		System.out.println("For Loop:");
		for (int j = 0; j < 5; j++) {
			System.out.println(j);
		}
	}
}
```
